## Que Verificar en una Custom Shop Usada

### Inspeccion Fisica

| Area | Que Buscar | Red Flag |
|------|------------|----------|
| **Headstock** | Logo Gibson correcto, inlays nitidos | Logo borroso, pegamento visible |
| **Serial Number** | Verificar en Gibson.com | No coincide con ano |
| **Binding** | Sin grietas, color uniforme | Amarillento desparejo, despegado |
| **Frets** | Desgaste parejo, sin planos | Trastes muy gastados en posiciones 1-5 |
| **Truss Rod** | Funcional, cubierta intacta | Stripped, no ajusta |
| **Electronica** | Potenciometros suaves, sin ruido | Crackle, cortes de senial |
| **Finish** | Checking normal en nitro es OK | Burbujas, repintado |
| **Peso** | 8.5-9.5 lbs ideal | Mas de 10 lbs es pesado |

### Documentacion Requerida

- [ ] **COA (Certificate of Authenticity)** - Critico para Custom Shop
- [ ] **Hang tags originales** - Aumenta valor de reventa
- [ ] **Case candy** - Llave allen, pano, etc.
- [ ] **Estuche original** - Brown/pink Custom Shop case
- [ ] **Recibo original** - Util pero no critico

### Verificacion de Serial Number

1. Ir a: [Gibson Serial Number Lookup](https://www.gibson.com/en-US/Support/Serial-Number-Search)
2. Ingresar serial number
3. Verificar: ano, modelo, lugar de fabricacion
4. Custom Shop: debe decir "Nashville" o "Custom Shop"

## Comparativa: Nueva Standard '60s vs Custom Shop Usada

| Aspecto | Standard '60s Nueva | Custom Shop Usada |
|---------|---------------------|-------------------|
| **Precio** | $2,599 + tax | $3,500 - $5,000 |
| **Calidad construccion** | Excelente | Premium/artesanal |
| **Maderas** | AAA grade | Hand-selected premium |
| **Pastillas** | Burstbucker 61 | Custombucker/PAF replicas |
| **Acabado** | Nitro standard | Nitro premium, hide glue |
| **Perfil cuello** | '60s SlimTaper | Historicamente exacto |
| **Garantia** | Si, de fabrica | No (salvo que compres en tienda) |
| **Valor reventa** | Deprecia ~20% | Mantiene/aumenta valor |
| **Disponibilidad** | Facil encontrar | Hay que buscar |

### Recomendacion

- **Standard '60s Nueva:** Si quieres garantia, disponibilidad inmediata, y presupuesto mas ajustado
- **Custom Shop Usada:** Si buscas la mejor calidad posible, no te importa el riesgo, y quieres algo especial
