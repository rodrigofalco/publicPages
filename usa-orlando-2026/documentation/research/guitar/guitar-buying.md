## Alternativas al Gibson Les Paul

### Epiphone Les Paul Standard (~$500-$550 USD)

Epiphone es propiedad de Gibson y fabrica versiones mas accesibles.

| Aspecto | Gibson | Epiphone |
|---------|--------|----------|
| **Precio** | $2,599+ | $500-$550 |
| **Fabricacion** | USA | Asia |
| **Maderas** | Premium seleccionadas a mano | Calidad estandar |
| **Pastillas** | Burstbucker (~$250 el set) | ProBucker (mas economicas) |
| **Acabado** | Nitrocelulosa | Poliuretano |
| **Hardware** | Premium | Estandar |

**Veredicto:** La diferencia de precio es de aproximadamente $1,500-$2,250. Para musicos casuales, Epiphone ofrece excelente valor. Para profesionales o coleccionistas, Gibson justifica la inversion.

> "Con la linea Inspired by Gibson Custom y especialmente la Les Paul Custom, la brecha entre Gibson y Epiphone se ha reducido tanto como era posible."

### PRS SE McCarty 594 (~$899 USD)

El McCarty 594 fue disenado especificamente como alternativa a la Les Paul vintage de finales de los '50s.

| Caracteristica | Detalle |
|----------------|---------|
| **Concepto** | Construir una "vintage Les Paul" accesible |
| **Pastillas** | 58/15 LT 'S' - calidas, bajo output, dinamicas |
| **Ergonomia** | Mas facil de tocar que una Les Paul |
| **Versatilidad** | Push/pull para tonos single-coil |
| **Escala** | 24.594" (ligeramente mas larga) |

**Ventajas sobre Les Paul:**
- Mejor acceso a trastes superiores
- Ergonomia moderna superior
- Precio significativamente menor
- "Suena como una gran LP vieja, pero se siente como una nueva PRS"

### PRS SE Custom 24 (~$849 USD)

| Aspecto | Detalle |
|---------|---------|
| **Caracter** | Entre Les Paul y Stratocaster |
| **Mejor para** | Versatilidad maxima |
| **Sonido** | Mas moderno que una Les Paul |

## Informacion Adicional

### Impuestos en Florida

- **Sales Tax Florida:** 6% estatal + impuesto local (total ~6.5-7.5%)
- **Estimacion de impuesto:** $170-$210 USD sobre $2,599

### Transporte de la Guitarra

- **Estuche rigido:** Incluido con Gibson Les Paul Standard
- **En avion:** Puedes llevarla como equipaje de mano (cabe en compartimento superior)
- **Documentacion:** Guarda el recibo para declaracion en aduana

### Declaracion de Aduana Argentina

- Franquicia de viaje: $500 USD por persona
- Exceso: 50% de impuesto sobre el excedente
- **Estimacion de impuesto aduanero:** ~$1,050 USD sobre una guitarra de $2,599
